# Практична робота №7.0
## Масиви в JavaScript

### Опис
У даній роботі реалізовано систему бібліотеки книг.

### Функціонал:
- Виведення всіх книг
- Пошук за назвою та автором
- Фільтрація за жанром і рейтингом
- Топ-5 книг
- Статистика
- Операції з множинами (union, intersection, difference)

### Використані методи:
map, filter, find, sort, reduce

### Як запустити:
Відкрити файл index.html у браузері
